# Control-Dashboard
Responsive Admin Dashboard for an Ecommerce Website using HTML, CSS, and JavaScript with light and dark mode theme functionality.
Created using CSS Variables, CSS Grid, CSS Flexbox, Media queries for our responsive design, and CSS  transitions and animations for some cool animation effects.
